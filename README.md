# 🌊 AIML-RisingWaters
### Flood Prediction Using Machine Learning

## 📌 Project Overview

Floods are among the most devastating natural disasters, causing significant loss of life and property every year. This project uses Machine Learning to predict the likelihood of floods based on historical weather and environmental parameters.

The application provides a simple web interface where users can enter environmental data and receive an instant flood prediction.

---

## 🎯 Problem Statement

Traditional flood warning systems may not always provide timely predictions. This project aims to develop an intelligent flood prediction system that helps improve disaster preparedness using Machine Learning.

---

- 🌧 Flood prediction using Machine Learning
- 📊 User-friendly Flask web application

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/gayathri-114/AIML-RisingWaters.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
